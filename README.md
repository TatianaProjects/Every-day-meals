# Everyday Meals

Web application that helps users browse, search, and explore recipes for everyday cooking. It focuses on simplicity, usability, and a clean interface.

## Features

- Browse a variety of recipes.
- Search recipes by name or ingredients.
- View recipe details, including ingredients and instructions.
- Interactive UI with responsive design for desktop and mobile.

## Technologies Used

- HTML5 – for structure
- CSS3 – for styling and responsive layouts
- JavaScript – for interactivity and logic
